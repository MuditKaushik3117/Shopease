# ecommerce-platform
# 🛍️ ShopEase - Modern E-Commerce Platform

A modern full-stack e-commerce application built with **Next.js**, **Express.js**, **React**, and **Tailwind CSS**.

---

## 🚀 Features

- 🛒 Product Listing
- 🔍 Search Products
- 📂 Category Filter
- 📄 Product Details Page
- ❤️ Wishlist
- 🛍️ Shopping Cart
- 💳 Checkout
- 📦 Order History
- 👤 User Login & Registration (Demo)
- 👤 User Profile
- 📱 Responsive Design

---

## 🛠️ Tech Stack

### Frontend
- Next.js
- React
- Tailwind CSS
- TypeScript

### Backend
- Node.js
- Express.js

### Data Storage
- Local JSON (Backend)
- LocalStorage (Frontend)

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/shopease.git
```

### Install dependencies

#### Backend

```bash
cd backend
npm install
npm run dev
```

#### Frontend

```bash
cd frontend
npm install
npm run dev
```
